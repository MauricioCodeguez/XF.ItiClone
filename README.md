# Xamarin.Forms Iti Clone

Clone do aplicativo Iti desenvolvido em **Xamarin.Forms**.

Desenvolvido com o intuito de mostrar que conseguimos criar **interfaces bonitas usando Xamarin.Forms** e **sem pretensão de utilizar as interfaces em um aplicativo em produção**.

O que foi usado no desenvolvimento:
- Padrão MVVM (sem framework)
- [CardsView](https://github.com/AndreiMisiukevich/CardView)
- [PancakeView](https://github.com/sthewissen/Xamarin.Forms.PancakeView)
- [ZXing](https://github.com/Redth/ZXing.Net.Mobile)
- [XamarinEssentials](https://docs.microsoft.com/pt-br/xamarin/essentials/)
- [SkiaSharp](https://github.com/mono/SkiaSharp)
- XamarinForms 4.6.0.726
